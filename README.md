Project Title:
- Autonomous Arduino Robot

Description:
- A small autonomous robot built using an Arduino microcontroller, DC motors, and an L298N motor driver. The robot follows a predefined path and uses infrared sensors to detect walls and avoid obstacles.

Features:
- Autonomous forward, turning, and reverse navigation
- Infrared wall/obstacle detection
- Differential motor control using L298N driver
- Embedded C/C++ motor control code

Hardware Used:
- Arduino (Mega)
- L298N motor driver
- DC motors
- Infrared sensors
- Battery power supply

How It Works:
The Arduino controls two DC motors through the L298N motor driver. 
The program adjusts motor direction and timing to move the robot along a path while infrared sensors detect nearby walls or obstacles and adjust movement accordingly.
